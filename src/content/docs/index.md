## What is Nitrogen?

Nitrogen is a set of npm packages that solve recurring frontend infrastructure
problems: SEO, consent-aware analytics tracking, and compliance document
generation. Built for SvelteKit, distributed via private npm registry.

## Packages

- **@nitrogen/frontend-seo** — Meta tags, OpenGraph, JSON-LD, sitemap, robots.txt
- **@nitrogen/frontend-tracking** — Klaro CMP + Consent Mode v2 + service catalog
- **@nitrogen/frontend-legal** — Auto-generated Privacy Policy, Cookie Policy, Imprint
